Ce module permet de contrôler le volume système via S.A.R.A.H
Le plugin est basé sur l'outil en ligne de commande NirCmd.

  • Vous devez spécifier dans le fichier volume.prop la version de Windows que vous utilisez : 32 ou 64 (bits).
    • Exemple: "os_version" : "32"
  • Sarah coupe le son
  • Sarah remet le son
  • Sarah monte le volume
  • Sarah baisse le volume
  • Sarah met le volume a 50 % ("pour cent" étant optionnel)
  • Le réglage du son à un niveau précis ne supporte, pour l'instant, que les paliers de 10 allant de 0 à 100.
